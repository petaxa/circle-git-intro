## github cliのインストール

## gh auth login

- `github cliのインストール` と同じクリック式のセットアップスライド
- タイトルは「GitHub CLIでSSHログインする」
- クリックで以下の表示内容を切り替える
  - `gh auth login`
  - `What account do you want to log into?` をコード風チップで表示し、`GitHub.com` を選ぶ
  - `What is your preferred protocol for Git operations?` をコード風チップで表示し、`SSH` を選ぶ
  - ブラウザで認証する
- 次のハンズオン1で `gh auth status` を実行してログイン状態を確認する
